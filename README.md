# Johnny Dash V16 Premium

Versão visualmente mais próxima do layout desejado:
- sidebar lateral
- hero premium
- cards de patrimônio
- gráfico grande
- notícias
- bitcoin e dólar com análise mais detalhada
- clima com dias e se chove ou não
- Google Sheets com instrução de formato da planilha

## Variáveis na Vercel
DASHBOARD_PASSWORD=sua_senha
GOOGLE_SHEET_ID=cole_aqui_o_id_da_planilha

## Formato recomendado do Google Sheets

### Aba 1: investimentos
Tipo | Ativo | Código | Data da Compra | Valor Atual | Aporte Mensal | Rentabilidade | Observação

### Aba 2: custos
Data | Descrição | Segmento | Valor

### Aba 3: metas
Objetivo | Valor Alvo | Data Alvo

### Aba 4: aportes
Mês | Valor

## Deploy
1. Extraia o ZIP
2. Suba os arquivos em um repositório limpo no GitHub
3. Importe na Vercel
4. Adicione as variáveis
5. Deploy
